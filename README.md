# myproject

> 基于vue.js的仿支付宝导航条吸顶滑动效果组件，导航菜单数量和内容菜单数量可随意扩展，且导航菜单字数无限制，随意配置，如果顶部和底部有固定占位元素，也可直接配置，无需额外修改样式。

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev 后即可查看效果

```
